Top Ten HTML tags You Need To Build a Website

1. Paragraph: <p>

The paragraph tag instructs the code editor that the content you are writing in between the tags is long form. Such as this here.

2. Heading: <h1-h6>

There are six Heading tags that you can use: h1, h2, h3, h4, h5, h6. Headings are used for titles. <h1> is the biggest in size. Usually there is only one h1 used in a document. But that's your decision to make. Please keep in mind that you cannot or should not use an <h2> or any other lower heading until you use an <h1>.

3. Images: <img>

Images are important pieces of content for any website. Images are used to help the user get a visual understanding of what the website is about. This is the theory behind it. Adding images can be tricky because not only do you need to use the <img> tag but you need to include the path the computer needs to take to retrieve the image. Usually this is commanded via the src.Once you add the path, the image should appear on your website. Keep in mind that if your images are moved from their location, then you must update the root path.

4. Links: <a>

A website would not be a website without links, I'm just saying. The World Wide Web would not be magical if websites were static. Links are like clues to treasure maps of information. That information helps users find the solution to the problem they have. Adding links to a site is done using the anchor tag: <a>. Similar to <img> tag, the anchor tag needs to include a path to either an internal or external destination. The directions come with the assistance of a href, which stands for hyperlink reference. The hrefprovides the URLof the link, which then directs the broweser were to send the user.

5. Section: <section>

The <section> tag helps seperate a websites sections. Depending on the website, many websites have a lot of information. Sections bring clarity and organization to the chaos. You can think about <section> tag like chapters in a book. This is helpful for both Search Engine Optimization(SEO) strategy and for styling purposes.

6. Div: <div>

Unlike the <section> tag, that seperates diffenet content from one another, the <div>

unites. The <div> is the ultimate uniter for content. This tag helps bring content together into smaller blocks of information. Like <section>, the <div> can be seen as sub-chapters. Using this tag may not be sexy but its wicked awesome when you are trying to style your website. Another positive factor is that it helps to let the search engine crawlersindex the information.

7. Navigation: <nav>

Building a website is more than just building a website for the sake of building it. There has to be a reason. In other words, there is an intention. What is the reason? Most of the time people are online to find a solution to a problem. Some need inforamtion. Others may need to make a purchase. The point is that people are in need of something. When a user lands on a page they want to know where they can find what is that they are looking for. Usually in the Navigation you will find a menu. In that menu you will find options to navigate to internal links.

8. List: <:ul> and <ol> and <li>

Lists tags are very useful. Lists tags come in three's. What do I mean by that? In order to utilize the list tags you need to decide if the information you want to shere needs to be in order or it does not. If order is important that you would use an <ol>. If items on a list is not important than the <ul> tag is used.

The third list tag is the actual <li> tag. This is used when you list down your items regaedless of order. Each item on your list should inclide a <li>.

9. Body <body>

The <body> tag is used once in the HTMLdocument. This tag is important becasue this is where your entire content is placed. So, yes. The <body> is important.

10. DOCTYPE!

Before any code is typed, you need to first let the code editor know that the document you are about to create will be an HTML document. The DOCTYPE! declaration is the first command in any HTML document. The format is called a boilerplatewhich is the a formal way of setting up your text editor to recognize that the the content you will be working in is HTML compatible.

Conclusion

There you have it! These are the ten tags you need to build a website.So go ahead and start creating!
